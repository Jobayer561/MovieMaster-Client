## Project Brief

**MovieMaster** is a modern, user-friendly platform for movie lovers to discover, explore, and track movies from all genres. It offers essential movie details, my collections, watchLists, and seamless browsing.

The goal of this platform is to give users a simple yet powerful movie experience.

---

## Project Description

MovieMaster provides an engaging, smooth movie discovery experience:

* Browse movies with detailed pages.
* Add movies to your  collection.
* Maintain a Watchlist.
* Filter movies by rating and genre
* Enjoy a clean UI with animations.
* View movie stats and featured sections.

This platform helps users organize their movie interests while keeping the experience visually appealing and easy to use.

---

## Purpose

The main goals of **MovieMaster** are:

* Allow users to manage and track their favorite movies.
* Provide detailed movie information.
* Deliver a smooth and responsive UI/UX.
* Let users store personalized collections and watchlists.

---

## Features

* **Browse all movies**
* **Movie details page** (Cast, director, rating, plot summary, poster, etc.)
* **Add to Watchlist**
* **Add to Collections** 
* **Edit  movies**
* **Delete movies**
* **Featured movie slider (Swiper)**
* **Beautiful animations (Framer Motion)**
* **Fast loading with skeleton UI**
* **User authentication (Firebase)**
* **Toast notifications**
* **Fully responsive UI**
